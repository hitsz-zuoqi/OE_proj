download kinfu_remake, and install dependencies
https://github.com/hitsz-zuoqi/kinfu_remake

Set redwood_simulated as your datasets. This dataset can be download using tools in
https://github.com/hitsz-zuoqi/Open3D/tree/cuda/examples/Python/ReconstructionSystem/scripts
Simply run  ## download_redwood_simulated.sh## in your terminal, you can download the dataset.

I suggest you to download Open3d for mesh and point cloud visualization
https://github.com/hitsz-zuoqi/Open3D
For python, simply run ## pip3 install open3d##

There is also a document about how to use Open3d
http://www.open3d.org/docs/release/tutorial/ReconstructionSystem/index.html
